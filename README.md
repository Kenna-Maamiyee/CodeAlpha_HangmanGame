# 🎮 Hangman Game (Python)

## 📌 Description

This is a simple command-line **Hangman Game** developed using Python.
The program randomly selects a word, and the player must guess it letter by letter within a limited number of attempts.

---

## 🚀 Features

* Random word selection from a predefined list
* Tracks guessed letters
* Displays correct and incorrect guesses
* Limits the number of wrong attempts
* Win and lose conditions

---

## 🛠️ Technologies Used

* Python (Core concepts)
* Random module

---

## ▶️ How to Run the Program

1. Make sure Python is installed on your system
2. Clone the repository or download the file
3. Open terminal in the project folder
4. Run the following command:

```bash
python HangmanGame.py
```

---

## 🎯 Game Rules

* You must guess the word one letter at a time
* Each incorrect guess reduces your remaining attempts
* You win if you guess all letters before attempts run out
* You lose if you reach the maximum number of wrong guesses
